# Install Development Environment

```bash
# Install Hugo to Mac
brew install hugo

# Install Hugo to Windows
choco install hugo

# Clone site
git clone https://github.com/segurvita/HugoQuickStart.git

# Add Some Content
hugo new posts/my-second-post.md

# Start the Hugo server
hugo server -D
```
